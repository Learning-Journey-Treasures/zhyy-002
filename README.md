# 基于SpringBoot的中小医院管理系统

#### 介绍
基于SpringBoot的中小型医院信息管理系统，功能开发实现了部分的主要功能。就诊卡提供了手动和读卡两种方式录入，IC读卡器使用的是"德卡D3"，SDK使用的是德卡官网提供的。用户注册为了验证邮件的邮箱考虑到安全问题，暂不提供授权码，如有需求可使用自己邮箱，开启POP3/SMTP服务后，配置到自己项目里。
